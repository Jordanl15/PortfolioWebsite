Java Chat Program

This application allows users to connect to a server, send and receive messages, and manage server connections. The server manages multiple clients, broadcasting messages and maintaining a user list. The client side allows users to connect, change ports, and send messages using a graphical interface. Server-side features include client management, broadcasting user lists, and handling incoming messages. The application is built using Java with JFrame for the user interface and TCP/IP socket programming for communication.

Starting the server

The server interface includes a port configuration screen. Users can either select an existing port or specify a new one. Any newly specified port is saved locally and will be used by default in future sessions. Once the "Connect" button is pressed, the server begins running and is ready to accept client connections.

When a client connects, they are presented with the same port configuration interface. If the client attempts to connect with a mismatched port, the application prompts them to ensure the port settings align with the server. Upon successful connection, the client is asked to provide a username. Once connected, the server GUI updates to display the new client’s username and connection status.

Sending messages

After successfully connecting, the client gains access to the chat interface. Users can send and receive messages in real-time. The connected clients are displayed in a dedicated "Connected Users" section, while the chat area shows all messages along with the sender’s username. This ensures seamless communication among all connected clients.
