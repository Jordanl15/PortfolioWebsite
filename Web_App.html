UWP Chess Game

This .NET web application was developed using C#, the MVC design pattern, and an SQLite database to provide a robust and efficient solution for managing client projects. It features an intuitive interface that allows users to track item costs, maintain inventory, and view detailed project information. The application incorporates role-based access control, ensuring that users only have access to functionality aligned with their roles, such as administrators, managers, or standard users. Each role has specific permissions, enhancing both security and usability.

Additional features include dynamic reporting tools for project analysis and cost management, along with user-friendly forms for adding, updating, and managing data. The SQLite database ensures lightweight and efficient storage, making the application suitable for small to medium-sized businesses.

The home screen provides an overview of all available features, allowing users to quickly navigate between different sections such as clients, projects, and settings.

Clients

The "Clients" section lists all registered clients, providing an organized view of client details, ongoing projects, and associated costs.

The "Client Info" page displays detailed information about a selected client, including contact details, project history, and any outstanding balances.

Client Projects

The "Client Projects" section showcases all active and completed projects associated with a client. It includes timelines, cost breakdowns, and status updates.

The "Project Info" page gives an in-depth view of a specific project, detailing tasks, allocated resources, inventory usage, and total expenses. They can also be approved or denied for cost and project planning.
